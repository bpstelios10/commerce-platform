# GO PROJECT

this is a learning Go project.

Info about tech stack and architecture can be found in [TECH.md](TECH.md)

## Basic Execution

Format code: `go fmt`

Run code: `go run .`

Build code: `go build`

Run exec: `./commerce-platform`

### Products Service

Run with `go run ./services/products/cmd`

### Orders Service

Run with `go run ./services/orders/cmd`
